# Changelog

## 0.1.1 - 2025-11-22

- Fix: Converted SceneTracker UI to classic Application for Foundry v0.8 compatibility.
- Feature: Added debug logging setting and centralized logger.
- Feature: Added localization (lang/en.json) and replaced several UI strings.
- Chore: Added MIT LICENSE and initial manifest.json, updated module metadata + download URLs.
- Chore: Added CI workflow to validate repository on push/PR.

## 1.0.0 - 2025-11-22 (Draft)

- Release Candidate: initial stable release.
- Feature: Spend & Roll workflow — hub, overlay, chat button with pending and merged roll JSON.
- Feature: Theming engine with presets and custom per-slot overrides (client-scoped settings).
- Feature: Attack and spell integration; tool & skill roll targets supported.
- Integration: Midi-QOL and MATT adapters to support automation and trap triggers.
- Quality: Packaging, release tools, and CI validation; improved logging and runtime validation.
